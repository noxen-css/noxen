Introduction

Noxen CSS is an attribute-driven CSS framework. Instead of class names, you use nx="" attributes to style your components. This keeps your HTML clean, scannable, and semantic.

💡 Noxen uses nx="component" attributes instead of class names — making your markup cleaner and more readable.
<!-- Traditional CSS framework -->
<button class="btn btn-primary btn-lg">Click</button>

<!-- Noxen CSS -->
<button nx="btn" size="lg">Click</button>

Installation

Get started with Noxen CSS via npm, CDN, or direct download.

NPM

npm install noxen-css

CDN

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/noxen-css@2/dist/noxen.min.css">

<!-- JS (optional — for interactive components) -->
<script src="https://cdn.jsdelivr.net/npm/noxen-css@2/dist/noxen.min.js"></script>

Quick Start Template

<!DOCTYPE html>
<html lang="en" data-nx-theme="void">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/noxen-css@2/dist/noxen.min.css">
</head>
<body>
  <div nx="container">
    <h1>Hello Noxen!</h1>
    <button nx="btn">Get Started</button>
  </div>
  <script src="https://cdn.jsdelivr.net/npm/noxen-css@2/dist/noxen.min.js"></script>
</body>
</html>

AI Palette Engine

Generate a complete design system from a single hex color using Noxen.palette.apply().

Pick any color above and click "Generate" — Noxen derives surfaces, text, borders, and accents automatically.

// Generate palette from any color
const tokens = Noxen.palette.fromHex('#6366f1');

// Apply as a new theme
Noxen.palette.apply('#ff6b6b', 'my-brand');

// Preview without applying
const preview = Noxen.palette.preview('#10b981');

Token Engine & Export

Override, export, and bridge tokens to Swift, Kotlin, Figma, SCSS, or CSS.

Tokens are exported to the browser console. Open DevTools (F12) to see the output.

// Override tokens
Noxen.tokens.set('--nx-color-accent', '#ff6b6b');

// Export for any platform
const css = Noxen.tokens.export('css');
const json = Noxen.tokens.export('json');
const swift = Noxen.tokens.export('swift');
const figma = Noxen.tokens.export('figma');

Accessibility

Auto-ARIA injection, keyboard navigation, skip links, reduced motion support, and full page audits.

✅ Auto-ARIA: Noxen automatically injects role, aria-expanded, aria-checked, tabindex, and keyboard handlers on all interactive components.
