Noxen v2.1 — Feature Drop

Drawer · Command Palette · Carousel · Container Queries · Data Grid. All with CSS + JS + full API.

Feature 1 of 5

nx="drawer" — Slide-In Panel

Slide in from any edge — left, right, bottom, or top. One attribute. Full API. Zero layout shift.

<!-- Trigger --> <button data-drawer-open="my-drawer">Open</button> <!-- Drawer markup --> <div nx="drawer-backdrop"> <div nx="drawer" id="my-drawer" side="right"> <div nx="drawer-header"> <h3>Title</h3> <button nx="drawer-close">×</button> </div> <div nx="drawer-body">Content here</div> <div nx="drawer-footer"> <button nx="btn">Save</button> </div> </div> </div> // JavaScript API Noxen.drawer.open('my-drawer') Noxen.drawer.close() Noxen.drawer.toggle('my-drawer')
Feature 2 of 5

nx="command" — Command Palette

Press ⌘K anywhere on this page. Keyboard navigation, live search, grouped items.

Or press ⌘K / Ctrl+K — works globally on this page.

<div nx="command-backdrop" id="my-cmd"> <div nx="command"> <div nx="command-input-wrap"> <input nx="command-input" placeholder="Search commands..."> </div> <div nx="command-list"> <div nx="command-group"> <div nx="command-group-label">Navigation</div> <div nx="command-item"> <span class="cmd-icon">🏠</span> <span class="cmd-label">Go Home</span> <span class="cmd-shortcut">G H</span> </div> </div> </div> </div> </div> // JavaScript API Noxen.command.open('my-cmd') // Open Noxen.command.close() // Close // Cmd+K / Ctrl+K opens automatically — globally

Feature 4 of 5

nx-container — Container Queries

Components respond to their own container size — not the viewport. The future of responsive design.

Drag the resize handle → the card reflows based on its own width, not the page:

Container-Aware Card Responds to container
◎
Noxen Sphere X1

Stacks vertically when container is narrow. Stays horizontal when wide.

In StockNew
<!-- Wrap in a container --> <div nx-container="my-widget"> <!-- Layout changes based on container, not viewport --> <div nx-cq="stack-sm"> <!-- row on wide, column on narrow container --> <img> <p>Description</p> </div> </div> // nx-cq utility values nx-cq="stack-sm" row → column when container < 480px nx-cq="hide-sm" hidden when container < 480px nx-cq="show-sm" visible only when container < 480px
